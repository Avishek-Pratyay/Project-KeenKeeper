##Friendship Tracker App
#Project Name: Friendship Tracker

#Description:
       Friendship Tracker is a modern web application designed to help users manage and maintain               meaningful relationships. It allows users to keep track of their friends, monitor interactions,         and receive insights into their communication habits through an intuitive dashboard and                 analytics.

#Technologies Used:
       React.js – Frontend framework
       Tailwind CSS – Styling and UI design
       Recharts – Data visualization (Pie Chart)
       React Router – Page navigation
       Context API – State management
       JavaScript (ES6+) – Core programming

#Key Features:
      1. Friend Management
           View all friends in a clean dashboard
           Access detailed friend profiles
           Track relationship status (On Track, Due, etc.)
      2. Timeline Tracking
          Log interactions (Call, Text, Video)
          View interaction history with icons and filters
          Easily monitor communication patterns
      3. Friendship Analytics
          Interactive Pie Chart showing interaction types
          Hover to view counts of Call/Text/Video
          Visual insights to improve relationship consistency
    
#Conclusion:
          This project demonstrates practical use of React, state management, and data visualization to           build a real-world, user-friendly application focused on maintaining meaningful human                   connections.
